# SysmeLite PMS (Electron + React + SQLite)

## Requisitos
- Node.js 18+ (recomendado 20 LTS).

## Desarrollo (opcional)
```bash
npm install
npm run dev
```

## Generar instalador Windows (.exe)
```bash
npm install
npm run dist
```
El instalador se guarda en `dist/`.
